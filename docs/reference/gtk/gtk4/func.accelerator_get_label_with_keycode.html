Description [src]

Converts an accelerator keyval and modifier mask into a string that can be displayed to the user.

The string may be translated.

This function is similar to gtk_accelerator_get_label(), but handling keycodes. This is only useful for system-level components, applications should use gtk_accelerator_get_label() instead.

Parameters

display GdkDisplay*
 

A GdkDisplay or NULL to use the default display.

Accelerator keycode.

A newly-allocated string representing the accelerator.
