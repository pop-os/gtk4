Class

GdkDevice

Description [src] 

abstract class Gdk.Device : GObject.Object {
  /* No available fields */
}

The GdkDevice object represents an input device, such as a keyboard, a mouse, or a touchpad.

See the GdkSeat documentation for more information about the various kinds of devices, and their relationships.
